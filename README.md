# FAITHSTATE – Real Estate Web-Based Property Solution

## Project Overview

FAITHSTATE is a web-based real estate management system developed to digitalize and streamline property listings, agent–client interaction, and basic transaction tracking. The system is designed for both academic and real-world use, following standard web development practices to ensure scalability, security, and maintainability.

FAITHSTATE supports local development and production deployment using commonly adopted web technologies and tools.

---

## Technology Stack

**Frontend**
- HTML5, CSS3
- JavaScript (ES6+)
- React.js
- Vite (build tool)

**Backend**
- Node.js
- RESTful API architecture
- JWT-based authentication

**Database**
- PostgreSQL (or MySQL, depending on deployment)

**Tools**
- Git & GitHub
- npm
- VS Code

---

## Project Setup

### Prerequisites

Ensure the following are installed on your system:

- Node.js (v18 or higher recommended)
- npm (included with Node.js)
- Git
